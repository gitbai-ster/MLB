1. Ordinal encoder: рдпреЛ ordinal variables рдХрд╛ рд▓рд╛рдЧрд┐ рдЙрдкрдпреБрдХреНрдд рдЫ, рдЬреБрди categorical variables рд╣реБрдиреН рдЬрд╣рд╛рдБ рддрд┐рдиреАрд╣рд░реВрдХреЛ рдбреЗрдЯрд╛ рддрд╛рд░реНрдХрд┐рдХ рдХреНрд░рдо рдЕрдиреБрд╕рд░рдг рдЧрд░реНрджрдЫ, рдЬрд╕реНрддреИ рд╣рд╛рдореНрд░реЛ dataset рдорд╛ `Item Size` рд╕реНрддрдореНрднред рдпрд╕рд▓реЗ рдПрдХ mapping рд╕рд┐рд░реНрдЬрдирд╛ рдЧрд░реНрджрдЫ рдЬрд╕рд▓реЗ рдкреНрд░рддреНрдпреЗрдХ рд╡рд░реНрдЧрд▓рд╛рдИ рдПрдХ рд╕рдВрдЦреНрдпрд╛рд▓реЗ рдкреНрд░рддрд┐рдирд┐рдзрд┐рддреНрд╡ рдЧрд░реНрджрдЫ, рдЬреБрди рд╕реНрддрдореНрднрдорд╛ рд╡рд░реНрдЧрдХреЛ рдХреНрд░рдо рд╣реЛред

2. Categorical encoder: рдпреЛ nominal variables рдХрд╛ рд▓рд╛рдЧрд┐ рдЙрдкрдпреБрдХреНрдд рдЫ, рдЬреБрди categorical variables рд╣реБрдиреН рдЬрд╣рд╛рдБ рддрд┐рдиреАрд╣рд░реВрдХреЛ рдбреЗрдЯрд╛ рддрд╛рд░реНрдХрд┐рдХ рдХреНрд░рдо рдЕрдиреБрд╕рд░рдг рдЧрд░реНрджреИрди, рдЬрд╕реНрддреИ `Item Size` рдмрд╛рд╣реЗрдХрдХрд╛ рд╕рдмреИ featuresред рдпреЛ рдПрдХ one-hot encoding рд╣реЛ, рдЬрд╕рдХреЛ рдорддрд▓рдм рдкреНрд░рддреНрдпреЗрдХ рд╡рд░реНрдЧрд▓рд╛рдИ рдПрдХ binary рд╕реНрддрдореНрднрд▓реЗ рдкреНрд░рддрд┐рдирд┐рдзрд┐рддреНрд╡ рдЧрд░реНрджрдЫ: encoded variable рдмрд░рд╛рдмрд░ 1 рд╣реБрдиреНрдЫ рдпрджрд┐ pumpkin рддреНрдпреЛ Variety рдорд╛ рдкрд░реНрдЫ рд░ рдЕрдиреНрдпрдерд╛ 0ред

рддреНрдпрд╕рдкрдЫрд┐, `ColumnTransformer` рдкреНрд░рдпреЛрдЧ рдЧрд░реЗрд░ рдзреЗрд░реИ encoders рд▓рд╛рдИ рдПрдХрд▓ рдЪрд░рдгрдорд╛ рд╕рдВрдпреЛрдЬрди рдЧрд░рд┐рдиреНрдЫ рд░ рддрд┐рдиреАрд╣рд░реВрд▓рд╛рдИ рдЙрдкрдпреБрдХреНрдд рд╕реНрддрдореНрднрд╣рд░реВрдорд╛ рд▓рд╛рдЧреВ рдЧрд░рд┐рдиреНрдЫред
